# Gsc — Córtex OS

Consulta Google Search Console via API oficial (google-api-python-client). Lista properties, submete sitemaps, inspeciona URLs (canonical/indexacao/cobertura), gera relatorio de queries/clicks/impressions/CTR/position e monitora 301 redirects. Use quando o usuario mencionar google search console, gsc, search console, indexacao google, sitemap google, queries google, clicks google, impressions, ctr, posicao google, cobertura, url inspection, change of address, migracao seo. Tambem dispara com /gsc-cortex setup.

Skill do **Córtex OS**, sistema operacional de negócio open source para Venda Mais com Conteúdo.
